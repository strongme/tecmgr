第一步：注册登陆

注册成功后自动登入“教师成长记录系统”。再次登陆只需输入身份证号码和密码即可。

第二步：信息录入

点击>>基本情况>>基本信息后，呈现的是预览界面，点击页面右上角的“编辑”即可进行信息的录入与修改。

第三步：预览

需要任意点击一下其他项目再点击刚录入信息的那个项目才可以预览。
